description: Commit, push, and open a PR in one step
---
## Context
- Current git status: !`git status`
- Current git diff (staged and unstaged changes): !`git diff HEAD`
- Current branch: !`git branch --show-current`

## Your task
Based on the above changes:
1. Create a new branch if on main/master
2. Stage and commit with an appropriate message
3. Push the branch to origin
4. Create a pull request using `gh pr create` with a summary and test plan

Do all of the above. Do not do anything else.
